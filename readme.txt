The Stylista Theme is a WordPress Theme suitable for fashion websites, make-up artist shops, clothing line, hair stylists and other fashionista!
Stylista WP Theme was developed with Reverie (a theming framework for WordPress)  

Features

--  Responsive  - screen, tablet and mobile
 -- Customize options to Upload logo and favicon 
 -- Elementor page builder to help build or layout the site pages with rich content types
 -- Widgetized sidebar and footer
 -- Freedom to choose between full-width template and default page template
 --Ships with Customizable off-canvas menu plugin
-- Ships with page-order plugin
-- Supports SEO plugins for your website search engine optimization
